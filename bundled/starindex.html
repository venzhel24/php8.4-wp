PHP 8.4

Кочетов Даниил

Вступление

PHP 8.4 привносит множество новых возможностей и изменений. Мы рассмотрим основные нововведения, функциональность и изменения, которые ломают обратную совместимость.

Что нового

  • Функциональный стиль вызова new
  • JIT: новая реализация
  • Хуки свойств
  • Ассиметричный модификатор доступа
  • Функции для работы с массивами
  • Новые методы: array_find_key, array_any, array_all
  • Атрибут #[Deprecated]
  • Обновление BCMath: объектно-ориентированное программирование
  • Lazy load
  • Работа с графемами
  • Многобайтовые функции: mb_trim, mb_ucfirst, mb_lcfirst
  • Модули PDO и Reflection

Функциональный стиль вызова new

Теперь вызов конструктора может быть выполнен в функциональном стиле.

JIT: новая реализация

Обновление JIT для повышения производительности выполнения кода.

Хуки свойств

Добавлены хуки для управления доступом к свойствам.

Ассиметричный модификатор доступа

Позволяет устанавливать разные модификаторы для чтения и записи свойства.

Функции для работы с массивами

Добавлены новые методы: array_find_key, array_any, array_all.

Атрибут #[Deprecated]

Используется для указания на устаревшие элементы кода.

Работа с графемами и текстом

  • Поддержка работы с графемами
  • Новые функции: mb_trim, mb_ucfirst, mb_lcfirst

Изменения, которые ломают обратную совместимость

  • exit и die: новое поведение
  • Ужесточение типизации
  • Изменения в обработке DOM
  • Обновления opcache и JIT
  • Новые ограничения в функциях ext-dom

Выводы

PHP 8.4 предлагает значительное улучшение производительности, новые возможности для разработчиков и инструменты для более качественного кода. Однако изменения требуют осторожности при обновлении.
